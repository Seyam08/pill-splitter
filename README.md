## Installation

Install this app with npm

1. Clone the repository:

   ```
   git clone https://github.com/Seyam08/pill-splitter.git
   ```

2. Install the dependencies:

   ```
   npm install
   ```

3. Start the application:

   Development version

   ```
   npm run dev
   ```

   build command

   ```
   npm run build
   ```

   preview command

   ```
   npm run preview
   ```

4. Open the package.json file to see all the scripts.
